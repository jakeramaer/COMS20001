# game_of_life_cc_2019
Game of life visualisation (Graded at 70%) - Concurrent Computing 2019 (XTime Composer)

Initialises Game of Life on Xmos board - https://www.youtube.com/watch?v=C2vgICfQawE&t=222s

RUNNING THE MODEL

- Compiles and runs with XTime Composer. Able to process boards up to 1200x1200 at 150s per round.
